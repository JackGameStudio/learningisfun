# LearningIsFun 🏝️

一款让孩子主动背单词的工具。通过 PDF 上传、自动词库生成、间隔重复算法、游戏化岛屿建造，把记忆科学变成孩子的探险游戏。

## 功能

- 📄 **PDF导入** — 上传英语课本 PDF，自动提取单词
- 🔬 **自动拆解** — 词根/前缀/后缀智能识别
- 🗣️ **语音朗读** — 每个单词标准发音
- 📊 **间隔重复** — Leitner 5盒子科学记忆算法
- 🏝️ **岛屿建造** — 学习解锁建材，建造自己的词汇世界
- 📱 **移动端适配** — 手机/平板/电脑均可使用

## 使用方法

直接在浏览器打开 `index.html` 即可，无需安装，无需服务器。

## 技术栈

纯 HTML/CSS/JS，无框架，无后端，数据存储在浏览器本地。
