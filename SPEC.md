# LearningIsFun — 产品规格 v1.0

## 1. 概念与愿景

一款专为10岁孩子设计的英语单词学习工具，让"背单词"从痛苦的任务变成每天碎片时间里的"建材收集"游戏。孩子通过间隔重复记忆单词，积累的词卡解锁"词汇岛屿"上的建材，岛屿越大=词汇量越大。

## 2. 目标用户

- **学习者**：JACK 10岁儿子
- **管理者**：JACK（上传PDF、管理词库、查看进度）
- 使用场景：每天10-15分钟碎片时间，看心情
- 英语水平：学校正常水平，有基础
- 兴趣钩子：Minecraft、Roblox、短视频

## 4. 交互规格

### 4.1 刷词复习流程

```
首页 → 点"刷词" → 复习（卡片翻面+3按钮） → 学习新词（3-5个）
→ 测试（3-5题） → 奖励动画 → 返回首页
```

**复习卡片**：
- 正面：单词 + 发音按钮
- 点击"显示答案" → 翻面：中文释义 + 例句 + 词根拆解
- 3按钮：🟢记住了（间隔增加）/ 🟡有点忘（间隔不变）/ 🔴不会（打回盒子1）

**学习新词**：
- 卡片展示：单词 → 发音 → 词根拆解 → 例句 → 确认进入词库

**测试**：
- 题型A（4选1）：给单词，选中文意思
- 题型B（填空）：给中文 + 词根提示，写出单词
- 随机出现，各50%

### 4.2 PDF导入流程

```
上传PDF → 文字提取 → 单词去重 → 过滤常见词 → 词典查义
→ 词根拆解 → 例句获取 → 预览页 → JACK确认 → 导入词库
```

### 4.3 数据存储

- LocalStorage 键：`lif_vocabulary`（词库）、`lif_progress`（学习进度）
- 导出格式：JSON

## 5. 算法规格

### 5.1 Leitner 5盒子

| 盒子 | Minecraft风格 | 规则 | 复习频率 |
|-----|------------|------|---------|
| 1 | 🔴 圆石 | 不会 | 每次都复习 |
| 2 | 🟡 泥土 | 答对1次 | 每2天 |
| 3 | 🟢 橡木 | 连续答对 | 每4天 |
| 4 | 🔵 铁块 | 稳定掌握 | 每8天 |
| 5 | 🟣 绿宝石 | 永久词库 | 每14天 |

### 5.2 简化SM-2算法

```
初始：I=1天，EF=2.5

🟢记住：
  I = I × EF（最多30天）
  EF = EF + 0.1（最高3.0）
  n++
  box = min(5, box + 1)
  nextReview = today + I

🟡有点忘：
  I 不变
  EF = max(1.3, EF - 0.1)

🔴不会：
  I = 1天
  EF = max(1.3, EF - 0.2)
  box = 1
  n = 0
  nextReview = today
```

## 6. API规格

| API | 用途 | 限制 |
|-----|------|------|
| `api.dictionaryapi.dev` | 英文释义+例句 | 无key，免费 |
| Tatoeba API | 真实例句 | 每秒1请求，有缓存 |
| Web Speech API | TTS发音 | 浏览器原生，零配置 |
